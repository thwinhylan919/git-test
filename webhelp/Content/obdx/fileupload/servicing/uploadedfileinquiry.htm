UPLOADED FILES INQUIRY

Using this option you are allowed to view the bulk files uploaded by the customer. This function should provide customer to view status of bulk files uploaded. The search can be filtered on various parameters like status, transaction identifier. You can trace the status of the file and if there is an error in the file, you can also download the error file to arrive at the exact reason for error. You can also trace the history of the file. Individual record details can also be checked.

How to reach here:

Dashboard > Uploaded Files Inquiry
OR
File Upload > Uploaded Files Inquiry

To search and view uploaded files

  1. Click Uploaded Files Inquiry. The uploaded Files Inquiry screen appears with the file uploaded details.

The file life cycle / file status could be:

  • Uploaded: File Uploaded and file reference number is generated.
  • Verified: File has been pre-processed and authorization checks done (limit + account access check).
  • Error: File has been pre-processed and gets error. The end of the lifecycle of the file.
  • Verified: File has been pre-processed and authorization checks done (limit + account access check). File will be sent to host for processing if no approval is required.
  • Approval in Progress: File has been partially approved. File will be available for further approvals.
  • Rejected: File has been rejected (File level). The end of the life cycle of the file.
  • Approved: File has been fully approved.
  • Completed

FAQs

What are the different transaction types supported by the file upload module?

The various file upload transaction types supported are:

  • Internal funds Transfer, (Within the bank)
  • Domestic Funds Transfer (Within the Country)
  • International Funds Transfer (Cross Border)
  • Mixed Transfer (A combination of the above)

What are the accounting types file upload supports?

Following are the accounting types supported by file upload:

  • Single Debit, Single Credit (SDSC)
  • Single Debit, Multi Credit (SDMC)
  • Multi Debit, Multi Credit (MDMC)
