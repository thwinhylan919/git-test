Change Password

This feature allows the existing users of the bank to change their log in password when required.

How to reach here:

Dashboard > User Profile > Change Password

Change Password

Field Description

Field Name

Description

Please enter your new password

Old Password

Old password for channel access.

New Password

New password for channel access.

Re-enter New Password

Re-enter the new password to confirm.

To reset the password:

  • In the Old Password field, enter the password.
  • In the New Password field, enter the password.
    OR
    Click to view the password policy.
  • In the Re-enter Password field, re-enter the password.
  • Click Change Password.
    OR
    Click Cancel to cancel the transaction.
  • The success message of changing the password appears. Click Login on confirmation screen to log in to the application.
