Authorization - Entitlement

The Authorization Entitlement allows the bank administrator to:

  • Create Entitlement
  • View Entitlement
  • Edit Entitlement

Edit Entitlement

Using this option you can edit or update the details of an existing entitlement.

To edit an entitlement:

  1. Repeat step 2 of Search Entitlement section.
  2. Click of the record which you want to edit. The Update Entitlement section appears.
  1. In the Display Name field, enter the display name of the entitlement.
  2. In the Description field, enter the description of the entitlement.
  3. Click Apply.
    OR
    Click Cancel to cancel the transaction.
  4. The screen with success message appears. Click if you want to delete the record.

Delete Entitlement

Using this option you can delete an existing entitlement.

To delete an entitlement:

  1. Repeat step 2 of Search Entitlement section.
  2. Click for the record which you want to delete. The Delete Warning message appears.
  1. Click Confirm.
    The Entitlements screen with the successful object deletion message appears. Click Done to complete the transaction.
