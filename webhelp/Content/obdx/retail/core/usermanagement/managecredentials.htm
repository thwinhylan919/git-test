Manage Credentials

The bank administrator manages the credential related operations such as:

  • Reset Password - allows to reset the log in password

How to reach here:

Dashboard > User Management > Search User > Manage Credentials

To manage credentials:

  • In the User Management screen, click Search. The search result section appears.
  • Select the desired user and click the Manage Credentials link. The Manage Credentials section appears.
