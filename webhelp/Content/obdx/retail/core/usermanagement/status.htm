Status

Using this option, bank admin can modify and manage the locked and activated statuses of a particular user.

How to reach here:

Dashboard > User Management > Search User > Status

To change the user status:

  • In the User Management screen, click Search. The search result section appears.
  • Select the desired user and click the Status link. The Status section appears.
Field Description

Field Name

Description

Lock Status

The current status of the user.

The status can be:

  • Lock
  • Unlock
  1. In the Lock Status field, select the appropriate option.
  2. Click Confirm.
  3. The success message appears. Click Done to complete the transaction.
