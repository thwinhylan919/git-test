AUDIT

Using this option, bank Aadministrator can view the audit log which is maintained for any tasks or transactions accessed by users. This option facilitates access control and supervision.

How to reach here:

Admin Dashboard > Audit

Search Audit

Using this option, bank administrator . The search result displays a list of particular audit based on different search parameters.

If the search parameters are not specified, then it displays all the records under the particular user type.

Field Description

Field Name

Description

Audit Type

Type of audit for which details are required.

User ID

User id for which the audit details are logged.

Context Id

Context Id of the transaction.

Party Id

PartyA party is any individual or business entity having a banking relationship with the bank. Id of the logged in user for which audit details are logged.

Request URL

Request url for which audit is details are logged.

This field appears if you select REST option from Audit Type list.

Start Time

Start time of the request processing.

End Time

End time of the request processing.

Service Name

Service name of the host.

This field appears if you select HOST option from Audit Type list.

Operation Name

First name or given name of the user.

This field appears if you select HOST option from Audit Type list.

To view the audit logs:

  • Click Search.
  • The Audit screen with search results appears based on the search parameters. Click Reset if you want to reset the search parameters.
