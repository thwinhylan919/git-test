Limits Maintenance

The Limit Maintenance module allows the Bank AdministratorAdministrator is a set of individuals that administer the applicant/Affiliate entity. For example, Accountants, Authorized Signatories for organizations, Power of Attorney for individuals. to create limits package for specific transaction type.

These limit packages defines the minimum and maximum amount values per transaction as well as maximum and minimum number of transactions that can be initiated per day.

Using this you can perform the following actions:

  • Create Limit Package
  • View Limit Package
  • Edit Limit Package

How to reach here:

Admin Dashboard > Limits

Limits - Create Limit

Using this option the bank administrators define the limit package for a specific transaction.

How to reach here:

Admin Dashboard > Limits > Limits

To create a limit:

  • In the Limits screen, click the Events link. The Limits screen appears.
  • Click . The Create Limit screen appears.
Field Description

Field Name

Description

Limit Name

Name of the limit.

Limit Description

Description of the limit.

Is User Changeable

Indicates whether limit if changeable by user.

The options are:

  • Yes
  • No

Currency

Currency in which the limit is to be created.

Select Event

The facility to select a transaction for which to define limits

Minimum Transaction Amount

The minimum amount limit in which a transaction of the specific type can be initiated.

Maximum Transaction Amount

The maximum amount limit in which a transaction of the specific type can be initiated.

Limit Type

Type of limit.

The options are:

  • Transaction Limit
  • Duration Limit

Duration Type

Type of duration limit (daily, weekly, monthly, yearly).

This field appears if you select Duration Limit from the Limit Type list.

Maximum Transaction Count

Maximum count of transactions.

This field appears if you select Duration Limit from the Limit Type list.

Start Date

The start date for the limit.
  • In the Limit Name field, enter the name of the limit.
  • In the Limit Description field, enter the description of the limit.
  • From the Currency list, select the appropriate currency.
  • From the Limit Type list, select the appropriate option.

  • From the Start Date list, select the appropriate date.
  • Click Create.

    Click Cancel to cancel the transaction.
