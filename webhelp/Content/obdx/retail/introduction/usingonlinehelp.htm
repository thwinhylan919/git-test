ABOUT WEBHELP

The Online Help is the primary source of information for the Oracle Banking Digital Experienceapplication. It is a convenient way to look up information about a task that you are performing or a feature or dialog box that you would like to know more about.

Using the Help Toolbar

The Help toolbar appears on top of each page and provides the following tools.

  • Expand All: Click to expand all the togglers available on the current page and view the entire content.
  • Collapse All: Click to close all the togglers available on the page.
  • Remove Highlights: Click to remove the search highlight.

Topic Structure

Every topic page is structured as follows:

Topic Name

This section provides the name of the transaction page.

Introduction

This section provides a brief introduction about the transaction page.

Navigation

This section provides an access path to the transaction page in the application.

Procedure

This section explains the steps used to perform the transaction. Depending upon the nature of the maintenance, only the mandatory procedure is explained.

Field Description

This section provides a general explanation of the fields available in the transaction page.
