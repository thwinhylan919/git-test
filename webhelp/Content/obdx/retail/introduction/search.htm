SEARCH IN WEBHELP?

The Webhelp provides search options that facilitates you to find what you are looking for within the Help. It provides a site wide search to search for a term in the entire Help.

Search for a term in the entire webhelp

You can search for a term, a word, or a phrase in the entire Webhelp.

To search within the webhelp:

  1. In the Search box on the top-right corner of the page, type the term that you want to search in the Help.
  2. Click .
    A list of topics that contain part or whole search criteria appears.
  3. Click the appropriate record from the search results.
    The selected topic appears in the right pane with the search criteria highlighted.

Note: You can also enter the text in quotation marks if you want to match the exact search criteria. The search string entered in the quotation marks is not case sensitive. For example, "Account number".
