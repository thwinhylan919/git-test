ReportS

The report generation module allows the bank administrator to generate the reports.

How to reach here:

Dashboard > Report Generation > Report

Field Description

Field Name

Description

Report Generation

Report Type

Type of report.

Report type can be:

  • Panel Setup for a given party Id
  • Workflow Setup for a given party Id
  • Pending approvals for a given user
  • Initiated transaction for a transaction type in given date range
  • Files Uploaded for a customer in given date range
  • Wallets opened in a given date range

Frequency

Frequency of the report generation.

Report Output Format

CurrencyA generally accepted form of money, including coins and paper notes, which is issued by a government and circulated with in an economy. of the limit group.The format in which you want to generate the report.

The options are:

  • PDF
  • CSV

Report Parameters

Party Id

PartyA party is any individual or business entity having a banking relationship with the bank. Id of the corporate user.

This field does not appear if you select Wallets opened in a given date range option from Report Type list.

Start Date

Start date of the date range from which you want to generate the report.

End Date

End date of the date range up-to which you want to generate the report.

To generate a report:

  • From the Report type list, select the appropriate option.
  • From the Frequency list, select the appropriate option.
  • From the Report Output Format list, select the appropriate option.
  • Click Register.
  • The success message appear. Click Download Report to download the report either in CSV or PDF format.
