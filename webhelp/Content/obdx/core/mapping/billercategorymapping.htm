Biller Category Mapping

Using this option, bank administrators can maintain biller category. And also map the one or more biller to the biller category.

This option allows the bank administrator to:

  • Search Biller
  • Map Biller
  • Manage Category

How to reach here:

Dashboard > Biller Category Mapping

Search Biller

Using this option, bank administrators search the biller mapped under the selected biller category. The search result displays a list of particular biller along with the biller code mapped to the selected biller category.
