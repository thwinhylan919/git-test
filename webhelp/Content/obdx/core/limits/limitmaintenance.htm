Limits Maintenance

The Limit Maintenance module allows the bank administrator to create limits group for all transaction.

These limit groups defines initiation limit and authorization limit of a transaction that perform by the user.

Using this you can perform the following actions:

  • Limit Group - Create Limit Group, Search / View Limit Group, Mapped Events, Mapped Roles, Edit Limit Group, Delete Limit Group
  • Limits - Create Limit, Search/ View Limits, Delete Limits
  • Events
  • Service

How to reach here:

Dashboard > Limits

Limits

Create Limit

Using this option the bank administrators define the limit which can be mapped to specific transaction.

How to reach here:

Dashboard > Limits > Limits

To create a limit:

  • In the Limits screen, click Create. The Create Limits screen appears.
Field Description

Field Name

Description

Limit Type

Type of the limit.

The options are:

  • Transaction Limit
  • Duration Limit

Limit Name

Name of the limit.

Limit Description

Description of the limit.

Currency

Currency in which the limit is to be created.

Minimum Transaction Amount

The minimum amount limit in which a transaction of the specific type can be initiated.

This field appears if you select Transaction Limit option from the Limit Type list.

Maximum Transaction Amount

The maximum amount limit in which a transaction of the specific type can be initiated.

This field appears if you select Transaction Limit option from the Limit Type list.

Cumulative Transaction Amount

The cumulative transaction amount limit in which a transaction of the specific type can be initiated.

This field appears if you select Duration Limit option from the Limit Type list.

Duration Type

Type of duration limit (daily, weekly, monthly, yearly).

This field appears if you select Duration Limit from the Limit Type list.

Maximum Transaction Count

Maximum count of transactions.

This field appears if you select Duration Limit from the Limit Type list.

Start Date

The start date for the limit.
  • From the Limit Type list, select the appropriate option.
  • In the Limit Name field, enter the name of the limit.
  • In the Limit Description field, enter the description of the limit.
  • From the Currency list, select the appropriate currency.
  • If you select Transaction Limitoption as a Limit Type:
  • In the Minimum Transaction Amount and Maximum Transaction Amount field, enter the appropriate amount.
  • If you select Duration Limitoption as a Limit Type:
  • In the Cumulative Transaction Amount field, enter the appropriate amount.
  • From the Duration Type list, select the appropriate option.
  • In the Maximum Transaction Count field, enter the appropriate count.
  • From the Start Date list, select the appropriate date.
  • Click Create.
    OR
    Click Cancel to cancel the transaction.
  • The success message appears. Click OKto complete the transaction.
