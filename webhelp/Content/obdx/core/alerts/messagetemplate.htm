MESSAGE TEMPLATE

The option allows the bank administrator to define required parameters for each messages like Email, SMS, and Secure Mailbox that is to be sent to banks customers.

The following features are maintained:

  • Create Message Template
  • View Message Template
  • Edit Message Template

How to reach here:

Dashboard > Application Toolbar > My Account > My Alerts > Message Template
OR
Dashboard > Alerts Set Up > Message Template

Create Message Template

Using this option, the bank administrator define the message template.

To create an message template:

  1. Click the Create tab. The Create maintenance got enabled.
Field Description

Field Name

Description

Template Id

The facility to specify the ID of the template which is to be linked to the alert.

Template Name

The name of the selected template.

Destination Type

The delivery mode through which the alert is to be sent.

The options are:

  • Email
  • SMS
  • Secure Mailbox

Message Template Details

Subject

The subject as specified in the template.

Content

The content of the template.
  1. In the Template ID field, enter the ID of the message template to be created.
  2. In the Template Name field, enter the name of the message template to be created.
  3. From the Destination Type list, select the appropriate option.
  4. In the Subject field, enter the subject for the message template.
  5. In the Content field, enter the subject for the message template.
  6. Click Set Mask to set the attributes of message template.
    OR
    Click Clearto reset the entered data.
  7. Entered the relevant data as per the template.

Edit Message Template

Using this option, bank administrator can edit the details of a existing message template defined in the application.

To edit message template:

  1. Click the Edit tab. The Edit maintenance got enabled.
