Alerts Maintenance

The Alert Maintenance option allows the bank administrator to define required parameters for each alert that is to be sent to banks customers.

The following features are maintained:

  • View Alert
  • Create Alert
  • Edit Alert
  • Delete Alert

How to reach here:

Dashboard > Application Toolbar > My Account > Alerts > Alert Maintenance
OR
Dashboard > Alerts Set Up > Alert Maintenance

Create Alert

Using this option, the bank administrator define the alerts maintenance.

Field Description

Field Name

Description

Alert Maintenance

Type of alert maintenance.

  • View: to view the existing alerts
  • Edit: to modify the details of selected alert
  • Create: to create a new alert

To create an alert:

  1. Click the Create tab. The Create maintenance got enabled.
Field Description

Filed Name

Description

Alert Name

The unique name to be given to the alert which is to be created.

Event Description

Description of the event for which alert is to be created.

Activity Name

The name of the activity associated with the selected event.

Event Name

Event name associated with the selected event description.

Event Date

The date on which the alert being created will expire

Alert Type

The type of alert i.e. if the alert is a subscription based alert or is one that is sent by the bank by default

The options are:

  • Subscribed
  • Bank Notification

Alert Nature

The nature of the alert i.e. if it is critical or informational.

The options are:

  • Critical
  • Informational

Dispatch Type

The alert dispatch type maintained for the alert and event combination

Link Message Template

Recipient Category

The facility to select the category of recipients of the alert.

The values can be:

  • Party

Recipient Name

The facility to select the recipient to whom the alert is to be available.

The values can be:

  • Customer

Delivery Mode

The delivery mode through which the alert is to be sent.

The options are:

  • Email
  • SMS
  • Secure Mailbox

Template ID

The facility to specify the ID of the template which is to be linked to the alert.

Template Name

The name of the selected template.

Subject

The subject as specified in the template.

Content

The content of the template.
  • In the Alert Name field, enter the name of the alert.
  • From the Event Description list, select the appropriate option.
  • From the Event Date list, select the appropriate date.
  • From the Alert Type list, select the appropriate option.
  • From the Alert nature list, select the appropriate alert.
  • From the Dispatch Type list, select the appropriate option.
  • In the Link Message Template section, enter the relevant information.
  • Click Save.
  • The Review screen appears. Verify the details and click Confirm.
    OR
    Click Cancel to cancel the transaction.
  • The success message appears. Click Done to complete the transaction.

View Alert

Using this option, bank administrator can search and view the details of any alerts maintenance based on the search parameters. If the search parameters are not specified, records of all the alerts maintained in the application are displayed.

To search alert:

  • Click the View tab. The View maintenance got enabled.
Field Description

Field Name

Description

Alert Name

Name of the alert defined at the time of creation.
  • From the Alert Name list, select the appropriate alert.
  • Click Search.
  • The Alert Maintenance screen with search results appears based on the selected alert. Click Reset to reset the search parameters.
  • To view the details of a specific alert, click the record. The detailed Alert Maintenance screen appears.

Update Alert

Using this option, bank administrator can edit the details of a selected alerts maintenance defined in the application.

To edit an alert:

  1. Click the Edit tab. The Edit maintenance got enabled.
Field Description

Field Name

Description

Alert Name

Name of the alert defined at the time of creation.

Event Description

Description of the event.
  1. From the Alert Name list, select the appropriate alert that you want to edit. The Alert Maintenance screen with detail of selected alert appears.
  2. To edit the alert, click Edit. The Edit Alert screen appears.
  3. Update the required fields.
  4. Click Save.
  5. The Review screen appears. Verify the details, and click Confirm.
    OR
    Click Cancel to cancel the transaction.
  6. The success message appears. Click Done to complete the transaction.

Delete Alert

Using this option, you can search and delete an existing alert.

To delete an alert:

  1. Repeat steps 1 to 2 of View Alert section.
  2. To delete an alert, click Delete.
  3. The Delete Warning message appears, click Confirm.
  4. The Alert Maintenance screen with the successful object deletion message appears. Click Done to complete the transaction.
