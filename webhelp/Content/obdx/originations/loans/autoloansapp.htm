AUTO LOAN APPLICATION

An auto or vehicle loan is a personal loan taken to purchase a new or used vehicle. In an auto loan, the vehicle being purchased is considered as the collateral on the loan.

Following are the steps involved in the loan application cycle:

  • Loan Requirement: You can specify the amount to be borrowed, tenure, repayment frequency, and purpose of the loan.
  • Orientation: It is the summary view of the loan application form which describes the required data to be entered and approximate time required to process the application form.
  • Confirmation: It is the agreement before starting the loan application process.
  • Borrowing Power: You have to specify the income and expense details, depending on which your borrowing capacity is determined.
  • Application: You have to specify personal, professional and financial details of all applicants, as well as vehicle details.
  • Offers: It displays multiple loan offers with an option to select any offer of choice.
  • Review: Summary of the loan application after selecting an offer and allows to edit details specific to any section.
  • Approval: Appears once the loan application is submitted successfully.

How to reach here:

Dashboard > New Auto Loan

To apply for loan:

  1. From the product selection screen, select the appropriate auto loan product. The Loan Requirement screen appears.
  1. In the Loan Amount field, enter the loan amount.
  2. From the Purpose list, select the purpose of the loan.
  3. From the Tenure Years list, select the loan tenure in years.
  4. From the Months list, select the loan tenure in number of months.
  5. From the Repayment Frequency list, select the loan repayment frequency.
  6. From the Co-applicant list, select the number of co-applicants.
  7. If you select one or more than one co-applicant, from the Co-applicant 1 relationship list, select the relationship with the primary applicant.
  8. To accept the terms and conditions, select I accept the Terms and Conditions check box.
