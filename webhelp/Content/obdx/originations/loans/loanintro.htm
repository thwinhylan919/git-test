PERSONAL LOAN ORIGINATIONS

The Oracle Banking Digital Experience Personal Loan Originations allows application of process for auto loans and unsecured personal loans. The applicant specifies account preferences, view the application form and track the status history of the application from the application tracker.

  • Auto Loans: Oracle Banking Digital Experience enables applicants, both existing customers of the bank as well as prospective, to apply for auto loans. The auto loan application captures basic details of the applicant/s such as personal, contact, occupation, financial details including details of income, expenditure, assets and liabilities as well as details of the vehicle being purchased. Applicants will be able to track the loan application once it is submitted, from the application tracker. In addition to tracking the progress of the application, the applicant will be able to complete pending tasks, if any, upload proof documents, and specify preferences concerned with loan repayment, application fees processing, view offer documents etc. Applicants can also retrieve a partially filled saved auto loan application and complete the same from the application tracker.
  • Unsecured Personal Loans: Existing customers as well as prospective customers can apply for unsecured personal loans. The application captures basic details of the applicant/s such as personal, contact, occupation, financial details including details of income, expenditure, assets and liabilities. Applicants will be able to track the loan application once it is submitted, from the application tracker. In addition to tracking the progress of the application, the applicant will be able to complete pending tasks, if any, upload proof documents, and specify preferences concerned with loan repayment, application fees processing, view offer documents etc. Applicants can also retrieve a partially filled saved auto loan application and complete the same from the application tracker.
